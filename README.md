# 🚆 IRCTC Redesign UI

A modern and responsive redesign of the IRCTC (Indian Railway Catering and Tourism Corporation) website built for UI/UX practice and frontend development learning.

## 📌 Project Overview

This project is a complete frontend redesign of the IRCTC website with a modern user interface, improved user experience, and responsive design. The goal was to reimagine the existing platform using modern web technologies while maintaining the core railway booking functionality layout.

## ✨ Features

- 🎨 Modern and clean UI design
- 📱 Fully responsive layout
- 🚆 Train booking interface
- 🚌 Bus booking section
- ✈️ Flight booking section
- 🏨 Hotel booking section
- ⚡ Smooth animations and transitions
- 🔍 User-friendly navigation
- 🌙 Modern color palette inspired by Indian Railways

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- JavaScript (ES6+)

### UI Components
- Lucide React Icons
- Custom Reusable Components

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/your-username/irctc-redesign.git
```

### Navigate to Project

```bash
cd irctc-redesign
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build Project

```bash
npm run build
```
